# CRUD API Project

A RESTful CRUD API built using FastAPI, SQLAlchemy, and SQLite.

## Features

- Create records
- Read records
- Update records
- Delete records
- SQLAlchemy ORM
- SQLite database
- Pydantic validation

## Project Structure

```
app/
├── config.py
├── crud.py
├── models.py
├── schemas.py
├── settlement.py
```

## Installation

```bash
git clone https://github.com/PriyaKunam7/simple-chatbot.git
cd simple-chatbot

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt
```

## Run

```bash
python create_tables.py
```
